# AI CV Career Advisor

An AI-powered career guidance web application for university students, fresh
graduates, interns, apprentices, and junior professionals. Paste or upload a CV
and receive **AI-generated** skill analysis, realistic entry-level role
recommendations, skill-gap insights, and practical learning suggestions.

> **Scope:** This tool provides AI-generated career **guidance only**. It does
> **not** connect to real internship or job listings and does not apply to
> positions on your behalf.

---

## Features

- Paste CV text **or** upload a **PDF / DOCX / image** CV
- Server-side text extraction (pypdf, python-docx, OCR) + cleaning
- AI analysis with **Qwen2.5-1.5B-Instruct** (run locally via llama.cpp)
- Structured report: professional summary, technical & soft skills, **exactly
  three** recommended roles, skill gaps, and a learning roadmap
- Per-role detail pages with suggested projects and a career roadmap
- **Copy**, **Print**, and **Download PDF** of the report (generated locally)
- Clear, honest empty/error states and a responsible-AI notice throughout

## Privacy

- **No database. No accounts. No permanent storage.**
- Uploaded CVs are processed only to extract text and are **not** saved.
- Results live in the browser's `sessionStorage` for the current session only.
- Message shown in-app: *"Your CV and analysis are not permanently stored by
  this application. Download your report before closing the session."*

## Architecture

```
┌─────────────────────────┐        HTTP (JSON / multipart)        ┌──────────────────────────┐
│   ai-cv-frontend        │  ──────────────────────────────────▶ │   ai-cv-backend          │
│   Next.js (App Router)  │   /analyze-text   /analyze-file       │   FastAPI + llama.cpp    │
│   TypeScript, Tailwind  │  ◀────────────────────────────────── │   Qwen2.5-1.5B (GGUF)    │
│   sessionStorage only   │        structured analysis JSON       │   pypdf / docx / OCR     │
└─────────────────────────┘                                       └──────────────────────────┘
```

- **Frontend** ([`ai-cv-frontend/`](ai-cv-frontend/README.md)) — Next.js UI. It
  validates input, calls the backend, adapts + validates the response, stores it
  in `sessionStorage`, and renders the report.
- **Backend** ([`ai-cv-backend/`](ai-cv-backend/README.md)) — FastAPI service
  that extracts CV text, runs the AI model, validates the output, and returns
  structured JSON. Temporary files are never persisted.

## Quick start

You need **two terminals** — one per service.

### 1. Backend (AI + extraction)

```bash
cd ai-cv-backend
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt          # first run downloads the ~1 GB model
uvicorn main:app --port 8000             # wait for "Qwen model is ready."
```

### 2. Frontend

```bash
cd ai-cv-frontend
npm install
cp .env.example .env.local               # defaults point at http://localhost:8000
npm run dev                              # http://localhost:3000
```

Open **https://ai-cv-career-advisor.vercel.app/analyze**, go to **Analyze CV**, and try the example CV.

## Responsible AI

The analysis is generated by AI and may contain mistakes. It should be used as
career guidance — **not** a hiring decision or professional certification.
Recommendations are explained using evidence from the CV; the app avoids
identity, personality, medical, or psychological judgments and does not show
fake accuracy scores, testimonials, or user counts.
